Block: Refers to an element that takes up the entire width of its container and starts on a new line, creating a block-like structure. Block-level elements typically stack vertically on top of each other.


Inline: Describes an element that does not start on a new line and flows within the text content. Inline elements occupy only the space necessary for their content and do not create line breaks.


Inline-block: Combines characteristics of both block and inline elements. An inline-block element flows within the text content like an inline element, but also allows you to set a width, height, padding, and margin properties like a block element. Inline-block elements do not break the flow of text and can be positioned horizontally next to each other
